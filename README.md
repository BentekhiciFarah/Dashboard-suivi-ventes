Dashboard conçu dans le cadre d’une formation que je suis pour apprendre à utiliser Power Bi dans l’analyse de données. Son objectif est d’offrir une vue d’ensemble du suivi des ventes d’une entreprise en centralisant les indicateurs clé de performance (KPI) sur une page, permettant ainsi à tout décideur d'analyser rapidement la santé commerciale de l'activité sans avoir à manipuler de fichiers bruts.
